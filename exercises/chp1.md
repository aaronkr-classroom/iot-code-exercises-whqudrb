# chp1 연습문제

1.Q1
  - (a) 디바이스의 센서와 액추에이터의 역할을 비교하여 설명하세요.
  -   센서와 액추에이터는 디바이스에서 핵심적인 구성 요소로, 각각 입력과 출력을 담당하며 상호 보완적으로 작동한다. 센서(sensor)**는 외부 환경의 물리적 변화를 감지하여 이를 전기 신호로 변환하는 장치이다. 온도, 빛, 압력, 습도와 같은 다양한 물리량을 측정하고, 측정된 데이터를 시스템으로 전달하는 역할을하는 입력 장치이다.
  -   액추에이터(actuator)**는 시스템으로부터 전달받은 전기 신호를 바탕으로 물리적인 동작이나 변화를 만들어내는 장치이다. 예를 들어 모터의 회전, LED의 점등, 스피커의 소리 출력 등을 수행하는 출력장치이다.
  - (b) 디바이스와 대시보드에서 동일한 프로그래밍 언어를 사용하지 않는 이유를 설명하세요.
  - (c) 브로커와 같은 정보 전달 전담 서버의 이점을 설명하세요.
